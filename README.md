# Spider Solitaire Backend

## Description
This project provides the infrastructure and a basic web server for managing game details. Leveraging Terraform for Infrastructure as Code (IAC), GitHub Actions for automated pipelines, and Node.js with Express for the web server, this backend serves as the backbone for the Spider Solitaire game.

## Link
[Spider Solitaire Backend](http://nodejs-env.eba-8buzdexv.eu-west-1.elasticbeanstalk.com/)

## Installation
1. Navigate to the `bootstrap` folder.
2. Run the following commands:
    ```
    terraform init
    terraform plan
    terraform apply
    ```
3. Proceed to the `terraform` folder and repeat the same commands.

## Usage
This backend is designed to be used in conjunction with the Spider Solitaire frontend. It handles game data storage and management.
